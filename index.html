Questionnaires

Aucun questionnaire pour l'instant. Créez le premier avec « Nouveau questionnaire ».

Écritures mathématiques : placez la formule entre deux symboles dollar — $\frac{3}{4}$ pour une fraction empilée, $5^2$ pour une puissance, $\sqrt{16}$ pour une racine. Un aperçu s'affiche sous le champ dès qu'un $ est saisi.

Classe

Chaque élève reçoit une carte numérotée. Le numéro imprimé sur la carte correspond à la ligne ci-dessous. Les fichiers de scan ne contiennent que les numéros, jamais les noms. Fichier CSV attendu (créé avec Excel) : colonnes numéro ; nom ; prénom ; niveau, avec une ligne d'en-tête.

Cartes-réponses

Une carte A5 par élève, avec un marqueur unique de 12 cm. L'élève tient sa carte avec la lettre de sa réponse (A, B, C ou D) vers le haut. Conseils d'impression : encre bien noire, qualité normale ou supérieure, pas de plastification brillante (les reflets empêchent la détection) — préférer une plastification mate ou du papier épais (160 g ou plus).

Projection

Sur le PC relié au vidéoprojecteur : cliquez « Ouvrir la projection », un code s'affiche. Sur le téléphone, en session de scan, entrez ce code (champ « Projeter »). La question et la liste des élèves (gris = pas encore répondu, bleu = a répondu) se mettent à jour en direct. Internet est nécessaire au moment de la connexion ; ensuite les données passent directement entre les deux appareils.
